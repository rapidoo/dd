# Campagnes D&D gratuites — Archive Markdown

Liste extraite et convertie depuis le fil D&D Beyond :
<https://www.dndbeyond.com/forums/dungeons-dragons-discussion/dungeon-masters-only/43718-list-of-free-dnd-campaigns>

Chaque entrée pointe vers un fichier `.md` local (texte extrait du PDF d'origine ou de la page web), ainsi que vers l'URL source.

> Conversion : `pdftotext -layout` pour les PDF ; extraction du markdown natif pour Homebrewery ; `pandoc` pour la page blog de *The Forge of Rogbrok*. Les PDF originaux sont conservés dans `_pdf/`.
